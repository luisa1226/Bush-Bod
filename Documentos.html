BUSH BOD

Objetivo General

Construir un vehículo que lea distancias para que asi este evite obstáculos a través de un sensor ultrasónico y con un control autónomo de si mismo.

Objetivos Secundarios

1. Elaborar planos esquemáticos, construcción y diseño para tener una idea clara del vehículo.

2. Realizar las conexiones electrónicas en simuladores a través de simuladores y así poder evitar cualquier contratiempo.

3. Revisar los debidos cálculos de voltaje, intensidad y resistencias que se puedan necesitar.

4. Aplicar lo visto anteriormente para comenzar con la construcción del vehículo.

5. Diseñar y seguir las indicaciones vistas y estudiadas en clase e internet para ejecutar correctamente el programa.

MATERIALES

1. Arduino Uno.

El arduino es el elemento fundamental ya que este nos permitirá programar el sensor y los motores.





2. Sensor Ultrasónico HC-SR04.

Este sensor nos permitirá hacer las mediciones correspondidas para su correcto movimiento.







3. Motor Servo.

Este motor nos permitira que el sensor de ultrasonido tenga un rango de medida de 180°.



4. Controlador L298 // Doble Puente H.

Este componente nos facilitara el control de los motores y la dirección que estos tomaran





5. Chasis coche.







6.Motor reductor

Este motor nos brindara la fuerza necesaria para mover el vehiculo.







Ficha tecnica de los materiales
